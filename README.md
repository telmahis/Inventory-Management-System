# 🏪 Inventory Management System (C++)

A console-based Inventory Management System built in C++ demonstrating core programming concepts such as file handling, data structures, input validation, and algorithm usage.

---

## 📌 Features

- 📦 Add new products with auto-generated unique IDs  
- 📋 Display all products in a formatted table  
- 🔍 Search products by ID or name (case-insensitive)  
- ✏️ Update product quantity and price  
- 🗑️ Delete products with confirmation  
- 📊 Sort products by price or quantity  
- 💾 Persistent storage using file handling  
- ⚠️ Low stock warning system  
- ✅ Robust input validation (safe user input handling)

---

## 🧠 Concepts Used

- C++ STL (`vector`, `algorithm`, `string`)
- File I/O (`ifstream`, `ofstream`)
- Exception handling (`try/catch`)
- Functions & modular programming
- Input validation using `cin.fail()`
- Lambda expressions (for sorting)
- Data parsing from files
